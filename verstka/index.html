Azure Marketplace

Магазин Azure Marketplace представляет собой каталог решений, сервисов, образов окружений разнообразной сложности, готовых для быстрого автоматического развертывания на платформе Azure.

Основная идея Azure Marketplace - дать независимому разработчику канал продаж своих решений по всему миру, а клиентам Azure - легкий способ развернуть фунциональные решения в собственной инфраструктуре.
